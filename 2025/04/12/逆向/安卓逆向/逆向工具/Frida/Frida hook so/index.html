Frida hook so

frida hook native常用模块

举例：

hello.so有printhello()方法

var pointer = Module.findExportByName(‘hello.so’, ‘printhello’)
var base_pointer = Module.findBaseAddress(‘hello.so’)
var module = Process.findModuleByAddress(base_pointer)

Module模块

.findExportByName(moduleName|null, exportName)

  • moduleName: lib名字
  • exportname: 函数名字
  • 返回exportName的地址指针

.getExportByName(moduleName|null, exportName)

  • moduleName: lib名字
  • exportname: 函数名字
  • 返回exportName的地址指针

.findBaseAddress(moduleName)

  • moduleName: lib名字
  • 返回lib的基地址

Process模块

.findModuleByAddress(address)

  • address: lib的指针地址
  • 返回一个Module对象

Memory模块

.readCString(pointer)

  • pointer:指针地址
  • 把pointer还原成字符串

.readUtf8String(pointer)

.readAnsiString(pointer)

Interceptor模块：监听

.attach(target, callback)

  • target: 指针地址
  • callbacks: 回调函数
    • onEnter
    • onLeave
      .replace(target, replacement)

查看加载了哪些SO文件

方式1: cat /proc/[pid]/maps

方式2：hook系统so库

  • 该库常调用函数：void * dlopen(const char *filename, int flag)；加载动态链接库

方式3： java.lang.Runtime

  • https://www.breakyizhan.com/java/4982.html
  • java.lang.Runtime：每个Java应用程序都有一个Runtime类实例
  • Runtime.getRuntime().loadLibrary() 加载lib

方案4：frida代码查看加载运行了哪些so文件
